## 📢 社区推广（如掘金、V2EX）核心卖点词

* **三零特性**：零服务器开销、零运维负担、零订阅费用。
* **白嫖配额**：每天 10 万次免费 Workers 调用，15 万条 D1 笔记行数，10GB 免费 R2 附件存储。
* **AI 大脑**：原生集成 Model Context Protocol (MCP)，将你的笔记瞬间转化为 Agent 的外部记忆库。
